= Getting JMX metrics with Jmxtrans

Jmxtrans is a way of obtaining JMX data from Java processes and pushing that data in various data formats to remote sinks
inside or outside of the cluster.
Jmxtrans can talk to a secure JMX port.
{ProductName} supports using the Jmxtrans to read JMX data from the Kafka brokers.

.Prerequisites
* A running kubernetes cluster
