= Logging

The `logging` field allows the configuration of loggers. These loggers are:

[source]
log4j.rootLogger
connect.root.logger.level
log4j.logger.org.apache.zookeeper
log4j.logger.org.I0Itec.zkclient
log4j.logger.org.reflections

For information on the logging options and examples of how to set logging, see xref:logging-examples-{context}[logging examples] for Kafka.
